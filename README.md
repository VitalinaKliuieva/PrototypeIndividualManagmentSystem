# Prototype: Individual Management System

## Overview

The Individual Management System is a web-based prototype designed to support users in managing their daily tasks, long-term goals, and scheduled activities within a single integrated platform. The system combines key productivity features such as task management, goal tracking, calendar scheduling, notifications, and analytics to provide a unified and user-friendly experience.

This prototype was developed as part of a software engineering project to demonstrate system design, object-oriented architecture, and user interface modeling.

---

## Objectives

The main objectives of this prototype are:

- To provide a centralized platform for managing tasks, goals, and events
- To improve personal productivity and time management
- To connect daily activities with long-term goals
- To visualize progress through analytics and dashboards
- To demonstrate a scalable and modular system design

---

## Features

###  Dashboard
- Overview of tasks, goals, and upcoming events
- Displays notifications and key productivity metrics
- Central navigation point for the system

###  Task Management
- Create, update, delete tasks
- Assign priorities (high, medium, low)
- Set deadlines and track status (pending, in progress, completed)
- Filter and sort tasks

###  Goal Tracking
- Create and manage personal or professional goals
- Link tasks to goals
- Track progress using visual indicators (progress bars, percentages)

###  Calendar System
- Visual monthly calendar view
- Schedule tasks into time slots
- Drag-and-drop task scheduling (prototype concept)
- View daily event details

###  Notifications
- Automatic reminders for deadlines and events
- Alerts for completed or upcoming tasks

###  Analytics
- Monitor task completion rate
- Track goal progress
- Provide insights into productivity

---

##  System Architecture

The system follows a modular architecture divided into the following subsystems:

- User Management
- Task Management
- Goal Tracking
- Calendar System
- Notification System
- Analytics and Reporting

Each subsystem is designed using object-oriented principles and interacts through well-defined interfaces.

##  Prototype Design

This project focuses on **UI/UX prototyping** and system specification rather than full implementation.

The interface includes:
- Dashboard view

The design follows:
